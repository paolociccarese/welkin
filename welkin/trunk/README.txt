


                                W E L K I N 




  What is this?
  -------------

  Welkin is an RDF graph visualizer.
  


  Requirements
  ------------

  Welkin is a command line application or an applet and requires:
  
   o  A Java 1.4 or later compatible virtual machine for your operating system.
